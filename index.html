Problem & Insight

Users face anxiety and uncertainty when sending on-chain transactions. Wallets expose raw blockchain metrics, not human-friendly guidance. The key insight: users need clarity, prediction, and reassurance, not raw data.

Solution

Fee Advisor AI uses on-chain signals + a prediction engine to suggest the appropriate fee tier (Slow / Normal / Fast / Turbo). It explains network conditions in plain language and warns users when a selection is likely to fail.

AI Reasoning
Translates mempool & base/priority fee data into human-friendly explanations.
Prediction
Outputs ETA, risk score, and a safety recommendation for the user.

High-Fidelity Screens

Dual-mode screens demonstrate consistent system behavior across dark & light themes.

System Architecture

Network → AI Logic → UI Recommendation → Wallet Signer → Blockchain

Network Layer
RPC Nodes, Mempool listeners, Gas Oracles
AI / Prediction
Model-based time estimate, risk detector, explanation generator
Wallet Integration
Signing via mobile wallet, deep links or extension

This work delivers a measurable improvement in user confidence and predictability when sending transactions. It’s a full-stack UX solution for Web3 payments.
